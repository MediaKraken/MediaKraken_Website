How the MediaKraken system works.





Installation and server setup

Decide if you wish to run an OVA on VMWare, ProxMox, VirtualBox or simply run Docker and Docker Compose on your OS of choice.

  • Download an OVA for VMWare 5.x+ or a Docker Compose yaml file. These are available on the downloads page.
  • NOTE: Only the Docker Compose download is up to date for code. If using OVA run the "mediakraken_update.sh" script in /home/mediakraken which will download the newest code. User for OVA is "mediakraken" and a password of "metaman".
    • If using own OS, install Docker and Docker Compose. Must install version 1.12 or greater for the new swarm version.
  • In mediakraken directory edit the "hidden" .env file.
    • Setup the database host/user/password
      • If using the built-in database simply change DBPASS to whatever you like. (Random chars/numbers is good)
      • If wish to use external postgresql 9.5+ database; change host, database, user, password
    • If using a MusicBrainz account, change the BRAINZCODE (NOTE: This is not used yet!)
  • In mediakraken directory run "docker-compose up -d" or can simply run "mediakraken_start.sh" if running on Linux host.
  • Browse to the host that you're running the server on port 8900. Ex. "http://hostname:8900"
  • Setup initial account. This account will automatically be set to an administrator account.
      • Click "sign-up" and register a new account.
      • Sign in with your new account
    • Setup your server environment
      • Click on "Server" in top main menu (this will only show up for administrators)
        • Shows tool links along with server information, notices, media counts along with stream and scan status's.
          • Add media via the Library link
          • After adding all media click the "Library Scan" button.
          • Scanning updates will show in the "Server" tab
