How is metadata managed by MediaKraken

  • theMovieDB
    • All metadata is stored in mm_metadata_movie as json.
    • Metadata is automatically udpated every 24 hours via theMovieDB API (updates and new records).
    • Metadata is downloaded individually for each media match during a media scan.
    • Metadata can be bulk downloaded via seperate program.
  • theTVDB and TVMaze
    • All metadata is stored in mm_metadata_tvshow as json at the "show" level
    • Episode data is within the actual show json
    • Metadata is automatically udpated every 24 hours via theTVDB API and TVMaze API (updates and new records).
    • Metadata is downloaded individually for each media match during a media scan.
    • Metadata can be bulk downloaded via seperate programs.
