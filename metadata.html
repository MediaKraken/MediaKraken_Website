How media matching works in MediaKraken.





How is metadata managed by MediaKraken

  • Movies
    • All metadata is stored in the database as json.
    • Metadata is automatically udpated every 24 hours via API (updates and new records).
    • Metadata is downloaded individually for each media match during a media scan.
  • TV
    • All metadata is stored in the database as json at the "show" level
    • Episode data is within the actual show json
    • Metadata is automatically udpated every 24 hours via API (updates and new records).
    • Metadata is downloaded individually for each media match during a media scan.
