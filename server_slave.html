MediaKraken Multimedia Suite - How it works





How the MediaKraken server slaves works.





What the slave server(s) do


  • Run as many as you want via Docker Swarm mode.
    • The server will fire up a new instance for each media file selected to play
    • This will allow Docker to load balance itself across the cluster.
  • This will do transcoding via FFMPEG or streaming if told to by the central server.
