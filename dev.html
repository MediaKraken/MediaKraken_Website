MediaKraken Multimedia Suite - Development





MediaKraken application suite development





Development

Interested in helping out? The bulk of the source code is Python 2.x at the moment and all source code is available on Github. Running your PR's through Pylint and creating unit tests for PyTest is greatly appreciated. All current unit tests are within the MediaKraken_Testing repository. Please see about page for module list and 3rd party libraries used in this application suite.


  • MediaKraken_Build - Tools/scripts/data to build database and OS
    • Python
  • MediaKraken_Common - Common code shared by tools/server/theater and OS.
    • Python
    • C
  • MediaKraken_Kodi - Plugin for Kodi
    • Python
  • MediaKraken_OctMote - Remote control program for phone and tablets
    • Python
    • Kivy
    • Plyer
  • MediaKraken_Roku - Roku Client
    • Brightscript
  • MediaKraken_Server - Main server component
    • Python
    • Twisted
    • Flask
  • MediaKraken_Slave - Transcoding slave for server
    • Python
    • Twisted
  • MediaKraken_Testing - Unit tests for all applications via PyTest and Selenium
    • PyTest
    • Python
    • Selenium
  • MediaKraken_Theater - Main client component for phone, tablet and PC.
    • Python
    • Kivy
    • Plyer
  • MediaKraken_Tizen - Samsung Tizen client
    • HTML
    • Javascript
    • Bootstrap
  • MediaKraken_Website - This website
    • HTML
    • Javascript
    • Bootstrap
