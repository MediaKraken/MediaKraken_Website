Development

Interested in helping out? The bulk of the source code is Python 2.7.x at the moment and all source code is available on Github. Running your PR's through Pylint and creating unit tests for PyTest is greatly appreciated. All current unit tests are within the testing directory. Please see about page for module list and 3rd party libraries used in this application suite. Jenkins server showing PyLint and code coverage via unit testing here.
